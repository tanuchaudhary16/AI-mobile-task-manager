# AI-mobile-task-manager
A mobile-friendly Mini Task Management System using Google Sheets as a backend. It enables team members to view assigned tasks, track daily progress, and update status in real time. Includes recurring and one-time tasks, scoring for delays, and Claude AI integration for smart insights and task suggestions.
This project is a mobile-friendly Task Management System that helps teams manage daily, weekly, and one-time tasks efficiently. It uses Google Sheets as a backend and provides real-time updates through a simple web interface.

🚀 Features
✅ Backend (Google Sheets)

Store task details:

Task Name
Task Type (Daily / Weekly / One-time)
Assigned To
Planned Date
Completion Date
Status (Pending / Done)
Weekly score tracking for missed or late tasks


📱 Frontend (Web App)

Mobile-friendly design
Displays only tasks assigned to the logged-in user
Shows:
Recurring Tasks
One-time Tasks
"Mark as Done" updates task instantly in Google Sheets


🤖 AI Integration (Claude API)

Smart task suggestions
Highlights pending or overdue tasks
Improves productivity with AI insights

🛠️ Tech Stack
Frontend: HTML, CSS, JavaScript
Backend: Google Sheets API
AI: Claude API (Anthropic)


⚙️ How It Works


Tasks are stored in Google Sheets
Web app fetches and filters tasks by user and date
User can view and update task status
Updates are reflected in real-time
AI provides additional insights and suggestions
